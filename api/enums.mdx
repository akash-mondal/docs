## Channel

WebSocket subscription channels:

```rust
pub enum Channel {
    /// L2 orderbook updates
    Book,

    /// Price ticker
    Ticker,

    /// Trade executions
    Trade,

    /// OHLC candlesticks
    Ohlc,

    /// Instrument information
    Instrument,

    /// L3 order-level data
    Level3,

    /// Private: order fills
    Executions,

    /// Private: balance updates
    Balances,
}
```

## TimeInForce

```rust

    /// Good until canceled
    GoodTilCanceled,

    /// Fill immediately or cancel remainder
    ImmediateOrCancel,

    /// Fill entirely or cancel entirely
    FillOrKill,

    /// Expires at specified time
    GoodTilDate(DateTime),
}
```

## BackpressurePolicy

```rust

    /// Drop newest messages when full
    DropNewest,
```
